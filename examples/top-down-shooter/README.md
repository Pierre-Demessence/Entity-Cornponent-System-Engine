# Top-Down Shooter Example

Rung 4 prototype for `@pierre/ecs`: twin-stick arena combat with
continuous mouse aim, held-fire bullets, and enemy swarms.

## Audio Credits

- Music: [8 Bit Crisis](https://opengameart.org/content/8-bit-crisis)
- Sound effects: [Byte-Man SFX #1](https://opengameart.org/content/byte-man-sfx-1)

## Run Locally

```sh
npm run dev -w @pierre/ecs-example-top-down-shooter
```
